# mdiewer demo

This is a **terminal Markdown viewer** written in Go. It supports `inline code`, *emphasis*, and [links](https://example.com).

## Lists

- Render headings
- Wrap paragraphs
- Handle fenced code blocks
- Show simple tables

1. Read a Markdown file
2. Convert it to terminal-friendly output
3. Print it with ANSI styles

> Markdown in the terminal should be readable first and decorative second.

## Code

```go
fmt.Println("hello from mdiewer")
```
